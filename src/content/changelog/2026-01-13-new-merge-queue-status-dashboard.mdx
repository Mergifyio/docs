A redesigned experience for the merge queue is now available. What's new:

- Clearer visualization of batched and queued pull requests with distinct sections
- Enhanced batch states, including bisecting batch visualization
- Improved performance with optimized data fetching
Prefer the classic view? Use "Go back to the old experience" to switch anytime. Your preference will be saved.

We're excited for you to try it out!
