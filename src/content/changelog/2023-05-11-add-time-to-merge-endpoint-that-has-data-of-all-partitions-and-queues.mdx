date: 2023-05-11
tags:

description: "The API now offers a new endpoint that returns the `time_to_merge` statistics for all the partitions and queues in the repository."
---

The API now offers [a new endpoint](https://docs.mergify.com/api/#/operations/get_time_to_merge_stats_for_all_queues_and_partitions_endpoint_repos__owner___repository__stats_time_to_merge_get) that returns the `time_to_merge` statistics for all the partitions and queues in the repository.
