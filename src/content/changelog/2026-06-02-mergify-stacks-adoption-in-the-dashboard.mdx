Mergify Stacks now has a dashboard surface. The new **Setup** tab gives you install commands and an AI-coding-agent prompt you can hand straight to your team, while the new **Adoption** tab shows active stacks, which repos are using Stacks, and trends over time across your organization.

All actions still happen on GitHub or via the CLI; the dashboard is read-only. Stacks itself remains zero-setup: no GitHub App, no dashboard auth required to use it.
