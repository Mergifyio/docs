---
title: "Allow to specify merge_conflict_style for copy and backport actions"
date: 2024-05-28
tags:
  - Workflow Automation
description: "Set `merge_conflict_style: diff3` on `copy` and `backport` to use the diff3 style for conflict markers."
---

You can now specify the `merge_conflict_style` options to `diff3` to have the conflicting code created using this style.
