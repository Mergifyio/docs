title: "`unqueue` command alias removed"
date: 2026-05-05
tags:

description: "The deprecated `unqueue` command alias has been removed; use `dequeue` instead."
---

The `@Mergifyio unqueue` command no longer works. It was renamed to `@Mergifyio dequeue` in November 2023 and has been kept as an alias since then. If you still have automation or comments relying on `unqueue`, switch to `dequeue`.
