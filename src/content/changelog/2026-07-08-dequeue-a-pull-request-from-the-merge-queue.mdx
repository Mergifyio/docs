description: "Manually remove a queued pull request from the merge queue from the dashboard, with a confirmation dialog showing the impact."
---

You can now dequeue a pull request from the merge queue directly from its side panel in the dashboard. Before removing it, a confirmation dialog shows how many in-progress batches will be interrupted and how many other pull requests will be rechecked, so you understand the impact before acting.
