date: 2025-03-14
tags:

  - Merge Queue
description: "`autosquash` is now off by default for the `queue` action, so Mergify can rebase without impersonating the author."
---

The `autosquash` option of the `queue` action has been disabled by default. This allows Mergify to rebase pull requests without having to impersonate the pull request author.
