The Mitigation page is now available to everyone. It lists tests quarantined on the default branch (added manually or by auto-quarantine). You can enable auto-quarantine** **so Mergify automatically quarantines flaky tests when they fail repeatedly on the default branch, so they stop blocking merges while you fix or investigate them. The page shows which tests are quarantined and whether auto-quarantine is on or off.
