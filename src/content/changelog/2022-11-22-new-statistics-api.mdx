Mergify now exposes a [statistics API](https://docs.mergify.com/api/#/operations/get_average_time_to_merge_stats_endpoint_repos__owner___repository__queues__queue_name__stats_time_to_merge_get) that allows to retrieve various metrics about queues:

- time to merge
- check duration
- failure by reason
- queue checks outcome
