---
title: "Configure merge queue commit message format from the dashboard"
date: 2026-06-02
tags:
  - Merge Queue
description: "A new modal editor lets you configure the declarative commit_message_format for queue rules directly from the dashboard."
---

When editing a queue rule from the dashboard, you can now configure how Mergify formats merge commits — title source, body content, and trailers — through a dedicated editor. This replaces hand-writing the legacy `commit_message_template` and keeps queue rule configuration entirely within the UI.
