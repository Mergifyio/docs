---
title: "Unified \"Mergify Merge Queue\" check run"
date: 2026-04-21
tags:
  - Merge Queue
description: "A single \"Mergify Merge Queue\" check run on pull requests, providing a unified view of merge queue status."
---

Every pull request in a repository with merge queue enabled now displays a "Mergify Merge Queue" check run. This check consolidates queue status information (whether the PR is queued, waiting for conditions, or has been dequeued) into a single, consistent check.
