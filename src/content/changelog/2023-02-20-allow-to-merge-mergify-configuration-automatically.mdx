---
title: Allow to merge Mergify configuration automatically
date: 2023-02-20
tags:
  - Workflow Automation
description: "The `queue` and `merge` action now supports a new option named `allow_merging_configuration_change`."
---

The `queue` and `merge` action now supports a new option named `allow_merging_configuration_change`. When set to `true`, this allows the actions to automatically merge any pull request that touches the Mergify configuration.
